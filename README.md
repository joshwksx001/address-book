# Address Book

A simple address book application built with plain JavaScript, HTML, and CSS. All contact data is stored in memory — no backend or database is used.

## Features

- Add a new contact
- View all contacts in a table
- Search contacts by name
- Delete a contact by ID

## Tech Stack

- HTML
- CSS
- JavaScript (no frameworks or libraries)

## How to Run

1. Download or clone this repository.
2. Open `index.html` in any web browser.
3. No installation or setup required.

## Data Structure

Each contact is stored as an object inside a JavaScript array.

```js
{
  id:    1,
  name:  "Joshua Wee",
  phone: "012-3456789",
  email: "joshwksx001@gmail.com"
}
```
